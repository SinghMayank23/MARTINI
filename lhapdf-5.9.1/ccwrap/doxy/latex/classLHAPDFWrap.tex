\subsection{Detailed Description}

\item[\textbf{ Deprecated}]The class-\/based C++ wrapper on \doxyref{L\+H\+A\+P\+DF}{p.}{namespaceLHAPDF} will be retired in a forthcoming release of \doxyref{L\+H\+A\+P\+DF}{p.}{namespaceLHAPDF} in favour of the namespaced wrapper declared in {\ttfamily \doxyref{L\+H\+A\+P\+D\+F.\+h}{p.}{LHAPDF_8h_source}}. Please convert client code which uses the class interface to use the new interface instead. Typically, this will just involve changing the header include from {\ttfamily \char`\"{}\+L\+H\+A\+P\+D\+F/\+L\+H\+A\+P\+D\+F\+Wrap.\+h\char`\"{}} to {\ttfamily \char`\"{}\+L\+H\+A\+P\+D\+F/\+L\+H\+A\+P\+D\+F\+Wrap.\+h\char`\"{}}, changing any constructors to initialisation functions, and replacing {\ttfamily \doxyref{L\+H\+A\+P\+D\+F\+Wrap}{p.}{classLHAPDFWrap}} objects with an {\ttfamily \doxyref{L\+H\+A\+P\+DF}{p.}{namespaceLHAPDF}} namespace.
